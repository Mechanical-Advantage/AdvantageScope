# 🔢 טבלה

תצוגת הטבלה מיועדת להציג את שינויי הערכים הספציפיים עבור סט שדות בפירוט. להוספת שדה, יש לגרור אותו לתצוגה הראשית. למחיקת שדה, יש ללחוץ על לחצן ה-"X" ליד שמו. ניתן לסדר מחדש שדות על ידי לחיצה וגרירה.

<img src="/img/tab-reference/table-1.png" alt="Overview" />

**שורה חדשה נוצרת עבור כל שינוי בכל אחד מהשדות המוצגים.** שורות כפולות אינן מוצגות. לדוגמה, טבלה המציגה את מצב הרובוט במהלך משחק תיראה כך:

<img src="/img/tab-reference/table-2.png" alt="Robot state table" />

הזמן שנבחר בטבלה מסונכרן בכל הכרטיסיות. יש ללחוץ על שורה כדי לבחור אותה, או להרחיף מעל שורה כדי לצפות בה בתצוגה מקדימה בכל החלונות הקופצים הגלויים. לחיצה על לחצן ה-↓ עוברת לזמן שנבחר (או לזמן שהוזן בתיבה).
