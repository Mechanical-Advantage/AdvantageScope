# דיאגנוסטיקה Phoenix

AdvantageScope תומכת בהזרמה חיה של אותות ממכשירי Phoenix 6 **ללא כל הגדרה בקוד המשתמש**. דבר זה מאפשר דיבאגינג וכוונון קלים של מכשירי Phoenix תוך שימוש בממשק המוכר ובכל העוצמה של AdvantageScope:

- אפשרויות ויזואליזציה גמישות, כולל תמיכה בצירים מרובים ובשדות בדידים
- תמיכה מלאה בגרפים מודעי-יחידות, כולל המרת יחידות מרומזת ובלחיצה אחת ([תיעוד](/tab-reference/line-graph/units))
- תצוגה מקדימה חיה של כל הערכים בסרגל הצד לעיון קל
- תמיכה בשרטוט גארפי ותצוגה מקדימה של אותות ממכשירים מרובים בו-זמנית
- פענוח ערכי enum כמחרוזות קריאות לאדם (מצבי בקרה, סטטוס גשר, מצב מגנט CANcoder וכו')
- נתוני עזרה משולבים בסרגל הצד עם תיאורים ויחידות עבור כל אות
- ארגון היררכי של אותות, מקובצים לפי ערוץ CAN, מכשיר וסוג אות
- ניתוח נתונים מתקדם עם אפשרויות מובנות של אינטגרציה וגזירה ([תיעוד](/tab-reference/line-graph/#adjusting-axes))

:::tip
להתחברות, יש לבחור "אבחון Phoenix" בעת התחברות לרובוט או לסימולטור מסרגל התפריטים.
:::
