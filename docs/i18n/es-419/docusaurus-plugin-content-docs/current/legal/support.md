# Soporte

Para recibir soporte para AdvantageScope o AdvantageScope XR, utiliza uno de los métodos de contacto a continuación.

- Para solicitudes de funciones e informes de errores, [abre un issue](https://github.com/Mechanical-Advantage/AdvantageScope/issues) en GitHub.
- Para todas las demás consultas, contacta a software@team6328.org.
